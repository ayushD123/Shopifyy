
# Shopifyy

A shopping ecosystem where buyers and sellers collaborate together


## Authors

- [@ayushD123](https://www.github.com/ayushD123)



## Features

- Responsive
- Live previews
- Fullscreen mode
- Cross platform
- Product Catalog
- Catalog Can be edited by seller
- User friendly UI
- Animations
